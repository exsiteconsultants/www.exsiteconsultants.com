Datahub CRM

Exsite is involved in building an online service to replace an existing CRM platform. The new platform follows the GDS guidelines for style and layout, adapted to deliver a CRM. Development is user research driven, with regular user testing sessions to gain feedback and help fine tune the design and user interaction.

Initially an Exsite designer and developer worked with a user research to develop a minimal prototype using Node JS to try ideas and get a feeling for what the user needed, using a combination of Postgresql, DJango Rest Framework and Node JS/Sass/Nunjucks.

The prototype allowed rapid development of interface ideas and journeys. Once the requirements started to settle the frontend was developed to integrate with the legacy backend. The interface continued to evolve through regular fed by the department and user testing sessions.
